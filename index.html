JavaScript Router API Reference

Router

The Router class is responsible for managing the application state and handling navigation between different routes. It initializes with a set of routes and provides methods to navigate and handle the current route.

Route

The Route class defines a single route in the application. It contains the path, the HTML content to display, and any associated JavaScript files to execute when the route is activated.

Methods

Router Methods

constructor(routes): Initializes the router with a list of routes.

navigate(path): Navigates to a specified path and updates the application state.

init(): Initializes the router, sets up event listeners, and handles the initial route.

Route Methods

constructor(path, html, script): Initializes a route with a path, HTML content, and an optional script.
